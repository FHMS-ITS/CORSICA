About this Project


About

corsica is a research project at the Münster University of Applied Sciences. The project aims at collecting information from those parts of the Internet that are not directly accessible from the Internet. A home user’s Intranet is one example for this. These Intranets may host devices such as IP cameras, or Wifi-enabled baby phones or toasters. There is evidence that many of the devices contain critical security vulnerabilities such as well-known, hard-coded user credentials. Criminals have been targeting these hidden devices in the last months, e.g. to build the Mirai botnet which was responsible for several large scale attacks against organizations.
